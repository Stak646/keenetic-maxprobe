# Документация (RU)

## Режимы
- FULL (по умолчанию) — всё, включая чувствительные.
- SAFE — редактирует секреты в копиях, но сохраняет список мест.

## Профили
- diagnostic — быстрее
- forensic — глубже (больше /proc и сырых дампов)

## Как читать архив
`fs/` зеркалит абсолютные пути. Пример:
`fs/opt/etc/init.d/S80lighttpd` = `/opt/etc/init.d/S80lighttpd`

Секреты и где их скрыть: `analysis/SENSITIVE_LOCATIONS.md`
